There are 2 available Liberty workshops with topics about API Provider feature for both OpenAPI 2 & OpenAPI 3, API Requester feature for both OpenAPI 2 & 3, security for inbound connections and outbound connection, and more. 

You can request these workshop presentations or ask any questions about the material by reaching out to our team using the **Contact** section on the homepage.

## Available Workshops

- **Accessing zOS resources with REST APIs using IBM zOS Connect**  
  Workshop covering configuration best practices for Liberty    
  [View workshop details →](zcee/provider)

- **WebSphere Liberty on zOS Introduction to Liberty Security**  
  Introduction workshop covering a wide array of security topics related to Liberty  
  [View workshop details →](zcee/requester)
