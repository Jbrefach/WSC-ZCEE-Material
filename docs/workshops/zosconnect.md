There are 2 available z/OS Connect workshops with topics about API Provider feature for both OpenAPI 2 & OpenAPI 3, API Requester feature for both OpenAPI 2 & 3, security for inbound connections and outbound connection, and more. 

You can request these workshop presentations or ask any questions about the material by reaching out to our team using the **Contact** section on the homepage.

## Available Workshops

- **Accessing zOS resources with REST APIs using IBM zOS Connect**  
  Workshop covering REST APIs, Intro to z/OS Connect, API Provider feature for both OpenAPI 2 & 3 and inbound security configuration    
  [View workshop details →](zcee/provider)

- **Accessing REST APIs from zOS using IBM zOS Connect**  
  Workshop covering REST APIs, API Requester feature for both OpenAPI 2 & 3 and outbound security configuration    
  [View workshop details →](zcee/requester)
